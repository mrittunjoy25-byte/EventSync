# College Event Management System - Frontend

## Overview

This frontend uses React, Vite, Tailwind CSS, React Router, React Hook Form, Axios, React Toastify, and Chart.js.

## Setup

1. `cd frontend`
2. `npm install`
3. Create `.env` with `VITE_API_URL=http://localhost:5000/api`
4. `npm run dev`

## Notes

- The frontend includes candidate pages for login, register, student dashboard, and admin dashboard.
- API calls are configured through `/src/services/api.js`.
